Machine Fingerprinting


Introduction

Hello there! We are network security researchers at Lehigh University developing a new way to identify computers. In order to test our ideas, we need your help to collect data. The test will have your computer render a series of images and send the results to our server. The information sent to the server contains absolutely no person information. We have provided programs to automate the process for your use or you can follow the instructions in the do it yourself section.

We understand that letting strange code run on your computer isn't a great idea, thus all of our code is open source. All the automated programs do is download the browsers (if needed) and open the tests. They do not download anything permanent to your computer and can be simply deleted after the test is complete.

Legal Stuff:

This research project has been reviewed by the Lehigh University Institutional Review Board (IRB) (Reference #: 16/191 S) and does not involve human subject research.

Instructions, pick one:

Automated

You just need to download a file, run it, and enter the manufacturer of your computer in each browser when prompted:
  • If you are using Windows, download this file and double click it
  • If you are using Ubuntu, download this file, unzip it, and run "run.sh"
  • If you are using OSX, download this file, unzip it, and double click RunOSX

    • Make sure to shut down the browser when you finished one. If not, the third browser can't start automatically.
    For Windows Users, Do Not Shut Down the Terminal (the black box) Until you Finish 3 browsers and see your code
